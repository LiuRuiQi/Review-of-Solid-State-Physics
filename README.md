# Solid State Physics Q&A

本文采用Q&A模式，力求条理清晰，简明扼要。总结完固体物理相关考点之后感觉固物以概念考察居多，比较基础，以下问题都比较典型。为方便大家复习记忆，Q.md中只列出问题，没有答案。

 1. 晶体、准晶体、非晶体的区别和判断方法。

	晶体：具有点阵的旋转、平移、反射等对称性，有长程序。
	
	非晶体：原子排列无序，没有长程序，只有局部短程序。
	
	准晶体：晶体中只具有1，2，3，4，6重旋转对称，准晶体是具有5重旋转对称的固体。准晶体点阵的平移对称不存在，但是有长程有序结构。

	用X射线衍射可以判断。

 2. 晶体缺陷的实验判断方法。 

	晶体缺陷分为点缺陷，线缺陷和面缺陷。可用衍射的方法测定（电子衍射、中子衍射、X射线衍射）。

 3. 能带理论的基本假设；直接能带和间接能带的定义

	能带理论的基本假设：不考虑散射、碰撞、能级跃迁等情况，将晶格中电子的运动近似为自由电子的运动，并且周期性势场很弱，可以当做微扰来处理。
	
	冯雪老师的课上应该是没有提过所谓的“直接能带”的，另外半导体有“直接带隙和间接带隙”的区分，不过应该不是我们需要关注的点。

 4. 布洛赫定理的定义

	当势场有周期性时 V(r) = V(r+Rn) 时，波动方程的解满足 ψ(r+Rn) = e<sup>ik\*Rn</sup> * ψ(r)。故，ψ可以写成 ψ(r) = e<sup>ik\*r</sup> * U<sub>k</sub>(r), 其中U<sub>k</sub>(r)在晶格平移下保持不变。物理意义：Bloch电子波函数是受晶格周期函数调制的平面波。

 5. 手机的相机和闪光灯里面都有什么半导体器件，都是什么材料做的；硅可以做感光器件，那它可以发光吗？

	闪光灯和相机中都有感光器件和控制电路，这些器件大多包含晶体管，而晶体管的核心工作原理就是PN结，因此可以用硅、砷化镓等半导体材料制作；至于硅能不能发光，从网上查到的结果是可以发光的，目前多孔硅的发光也是研究的热点之一。

 6. 一维双原子链的光学支和声学支画一下，那几个特殊点的意义是什么？ 两支的物理意义是什么？

	图的话可以参考课件，这里列出几个比较关键的特殊点的物理意义：长波极限对应着波矢q趋向于0时的情况，晶格振动的最大频率是在光学支上取得的，最小频率是在声学支上取得的，波数q在布里渊区边界取值时发生布拉格反射，形成驻波，群速度为零。

	长光学支格波的物理意义：每个原胞内的原子做相对振动，频率较高。

	长声学支格波的物理意义：原胞原胞中的原子做整体运动，频率较低。

 7. Si是什么晶体结构的，导带价带相对位置，费米能级在哪？

	Si是金刚石结构，导带的能量高，价带的能量低，费米能级在导带和价带之间。

 8. Si掺杂成n型加入什么物质？费米能级怎么移动？

	加入施主杂质（如5价的磷），费米能级升高，向导带靠近。

 9. 迁移率单位是什么？

	迁移率μ : 单位电场作用下载流子的平均迁移速度，故单位是速度单位除以电场单位，为m<sup>2</sup> / (V*s)

 10. 为什么要掺杂，好处有哪些？

	掺杂之后可以人为控制半导体的导电特性，改变热平衡时载流子的类型和浓度（本征半导体很难控制）。

 11. 固体比热容是由什么决定的，反映了什么内部性质？

	经典理论认为：固体比热和温度、材料无关；量子理论认为：固体热容是晶格振动的宏观性质上最直接的一个体现。实际上，在室温或者更高温度时，固体热容近似是一个常数，在低温时，随温度的负3次方快速衰减至零。

 12. 金刚石的结构是什么样的？那它是什么布拉菲格子？

	金刚石结构是“金刚石晶格”(Diamond Structure, DIA)，不是布拉菲格子（面试问题中的陷阱）。可以看成FCC的套构。

 13. 堆积比的定义

	晶格中原子球体积占总体积的百分数。 （注意，这里“原子球体积”不是原子的真实体积）

 14. 德拜模型的定义

	德拜模型在爱因斯坦模型的基础上，考虑了晶格振动频率的分布，更好地模拟了晶体热容的变化规律，克服了爱因斯坦模型得到的热容在低温情况下下降过快的缺点。

 15. 倒格空间的定义

	倒格空间是正格子空间的傅里叶变换，与波矢空间在同一空间中。

 16. 记得几种布拉菲格子？ 原胞中包含几个原子？体心立方的倒格子是什么？

	常见布拉菲格子：简立方(SC)、体心立方(BCC)、面心立方(FCC)。根据定义原胞中只有一个原子。BCC与FCC互为倒格子。

 17. 金属半导体接触都有几种，区别？
	
	肖特基接触和欧姆接触。肖特基接触要求金属的功函数大于半导体的功函数，而欧姆接触一般要求金属的功函数小于半导体的功函数。
